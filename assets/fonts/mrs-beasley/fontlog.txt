FONTLOG for the Mrs Beasley font

This file provides detailed information on the Mrs Beasley Font Software.
This information should be distributed along with the Mrs Beasley font and any derivative works.

Basic Font Information

Mrs Beasley is a Unicode typeface that supports most languages that use the Latin script and its variants, and could be expanded to support other scripts.
